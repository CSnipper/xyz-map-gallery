XYZ Map Gallery is a WordPress plugin for creating interactive maps with custom markers and photo galleries. Perfect for websites that need to present locations, places, or objects visually, with rich metadata and photo support.

== Description ==
XYZ Map Gallery lets you create unlimited custom maps, add markers with metadata and icons, and assign photos to markers and places. Display maps and galleries using Gutenberg blocks, Elementor, WPBakery, or shortcodes. REST API is available for advanced integrations. The plugin is ready for translation (i18n) and offers optional WooCommerce integration for selling photos (future feature).

== Features ==
* Unlimited custom maps
* Custom markers (gallery_items) with metadata and icons
* Assign photos (photo_items) to markers and places
* Display maps and galleries via blocks, Elementor, WPBakery, or shortcodes
* REST API for developers
* Translation-ready
* Optional WooCommerce integration (future)

== Use Cases ==
* Tourist guides and city maps
* Museums and exhibitions
* Real estate listings
* Nature reserves, parks, trails
* Event maps (festivals, marathons)
* Educational projects
* Any site needing interactive, photo-rich maps

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/xyz-map-gallery` directory, or install via the WordPress plugin screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the XYZ Map Gallery menu in WP admin.
4. Create your first map and add markers.
5. Assign photos and configure display options.
6. Use blocks, shortcodes, or page builders to show maps and galleries.

== Frequently Asked Questions ==
= Is WooCommerce required? =
No, WooCommerce is optional and only needed for e-commerce features.

= Can I use this plugin with Elementor or WPBakery? =
Yes, both are supported.
